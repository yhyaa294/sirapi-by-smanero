## 💡 Tips untuk Generate

1. **Konsistensi Warna**: Selalu sertakan hex color code di prompt
2. **Style**: Gunakan "3D isometric", "modern UI", "dark mode"
3. **Quality**: Tambahkan "8K", "professional", "Dribbble quality"
4. **Background**: Selalu specify "dark navy background" atau "transparent"
5. **Mood**: "Industrial", "tech", "cyberpunk", "professional"

## 📁 Naming Convention

Simpan hasil generate dengan format:
```
smartapd_[section]_[element]_v[version].png

Contoh:
smartapd_hero_worker1_v1.png
smartapd_icon_ai_brain_v2.png
smartapd_apd_helmet_v1.png
```
